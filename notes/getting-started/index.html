Welcome to the Bonham Lab!

If you're new, you should make sure you complete the New Member Onboarding.

This wiki is built with Obsidian. Some of the pages are available on the web, but many will only be available when you install obsidian and install the vault locally.

Once you have a local copy, peruse the Protocols and the Learning Resources, and make sure you're familiar with the Code of Conduct.

Other useful starting points to get started with are the Coding and version control protocol, the Reading and organizing literature protocol, and the Meetings and other admin protocol.
